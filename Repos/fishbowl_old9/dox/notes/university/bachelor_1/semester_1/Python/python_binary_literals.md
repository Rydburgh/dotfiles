Powers of 2:

2 ** 10 = 1,024 ~ 1,000

K=1024 (units we use to describe memory)

kilo = 1,000 (units we use to describe hard drives)

2**20 = 1K * 1K = 1M

2*30 = 1K * 1M = 1G

2**40 = 1K ** 1G = 1T (tera)

2**50 = 1K * 1T = 1P (peta)

2**60 = 1K * 1P = 1E (exa)
